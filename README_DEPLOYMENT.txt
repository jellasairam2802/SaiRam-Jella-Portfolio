# Sai Ram Jella - Business Analyst Portfolio

This folder is ready for GitHub Pages, Netlify, or any static hosting service.

## Files
- `index.html` - premium dark-theme Business Analyst portfolio
- `assets/SaiRam_Jella_Business_Analyst_Resume.pdf` - downloadable resume PDF

## GitHub Pages steps
1. Create a new GitHub repository named `sai-ram-portfolio`.
2. Upload the full contents of this folder, including `index.html` and the `assets` folder.
3. Go to Settings → Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select branch `main` and folder `/root`.
6. Save.
7. Your link will look like:
   `https://YOUR-GITHUB-USERNAME.github.io/sai-ram-portfolio/`

## Netlify steps
1. Go to Netlify.
2. Drag and drop this full folder or upload the zip file.
3. Netlify will publish it and give you a link.
4. You can rename the site to something like:
   `sairamjella-portfolio.netlify.app`

## Add to resume
Use a clickable text link in the top contact line:

`Portfolio`

or use the full public URL after hosting.
